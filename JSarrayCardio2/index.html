.some()

The some() method executes a callback function onces for each element present in the array until it finds the element that returns a truthy. This method takes a callback function that takes three arguments:

  • the index(optional)

    • The index of the current element being processed in the array.

  • the array(optional)

    • The array some() was called upon.

  • this keyword(optional)

    • A value to use as this when executing callback.
