.some()

The some() method executes a callback function onces for each element present in the array until it finds the element that returns a truthy. This method takes a callback function that takes three arguments:

  • the index(optional)

    • The index of the current element being processed in the array.

  • the array(optional)

    • The array some() was called upon.

  • thisArgs(optional)

    • A value to use as this when executing callback.

The Return Value:true if the callback function returns a truthy value for at least one element in the array. Otherwise, false.

.every()

The every() method tests whether all elements in the array pass the test implemented by the provided function. It returns a Boolean value. This method takes in a callback function that takes three arguments:

  • the index(optional)

    • The index of the current element being processed in the array.

  • the array(optional)

    • The array every() was called upon.

  • thisArgs(optional)

    • A value to use as this when executing callback.

The Return Value:true if the callback function returns a truthy value for every array element. Otherwise, false.

.find()

  • the index(optional)

    • The index (position) of the current element in the array.

  • the array(optional)

    • The array find() was called upon.

  • thisArgs(optional)

    • A value to use as this when executing callback.

The Return Value:TThe value of the first element in the array that satisfies the provided testing function. Otherwise, undefined is returned.
